# GymSpotter
!! NODE.JS TELEPÍTÉS SZÜKSÉGES !!

BACKEND:	A weboldal működéséhez először nyissuk meg a GymSpotter_backend mappát Visual Studio Code-ban, majd a terminálba írjuk be a következő parancsot: node app.js. Ezzel elindul a backend szerver. Ha a node_modules mappa még nincs jelen, először futtassuk az npm install parancsot a szükséges csomagok letöltéséhez. URL: http://localhost:3000/

FRONTEND:	Ezután nyissuk meg a gymspotter_maint mappát Visual Studio Code-ban, és a terminálba írjuk be az npm run dev parancsot a frontend elindításához. Ha a node_modules mappa hiányzik, előbb futtassuk az npm install parancsot a szükséges csomagok telepítéséhez. URL: http://localhost:5173/
